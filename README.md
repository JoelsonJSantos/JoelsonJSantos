## 💻 Projeto em desenvolvimento

### Indica Alagoas

Plataforma web para conectar clientes a profissionais de serviços locais em Alagoas.

Funcionalidades do projeto:

- Cadastro de profissionais
- Categorias de serviços
- Perfil profissional com descrição, fotos e contato via WhatsApp
- Painel administrativo
- Sistema de aprovação de profissionais
- Interface responsiva
- Banco de dados com Supabase
