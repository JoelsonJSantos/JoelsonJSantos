## 💻 Produto em Produção

### Indica Alagoas
🔗 **Acesse o portal ao vivo:** [indicaalagoas.com.br](https://indicaalagoas.com.br)

Plataforma SaaS full-stack desenvolvida do zero (MVP entregue em um *sprint* de 25 dias) para conectar clientes a profissionais de serviços locais, com foco em SEO, performance e usabilidade (Component Driven UI). 

**Principais destaques técnicos:**
- Arquitetura baseada em Next.js (App Router) com renderização otimizada e TypeScript para segurança de tipagem.
- Modelagem de banco de dados relacional e sistema de autenticação segura via Supabase (PostgreSQL).
- Integração de IA Generativa no fluxo de desenvolvimento para escalar a entrega de features.
- Pipeline ativo de CI/CD (Vercel/GitHub Actions) com histórico de versionamento rigoroso e dezenas de deploys em produção.
- Painel administrativo autônomo com moderação de perfis e gateway otimizado para o WhatsApp.
